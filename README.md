# Sudoku Solver with Backtracking Algorithm

This project is a Sudoku solver implemented using the Backtracking Design and Analysis of Algorithms (DAA) technique. It features both a logical core for solving and generating puzzles and a graphical user interface (GUI).

## Features

- **Backtracking Solver**: Efficiently solves standard 9x9 Sudoku puzzles.
- **Puzzle Generator**: Automatically creates new Sudoku puzzles for the user.
- **Graphical User Interface (GUI)**: Built using Python's `tkinter` for an interactive user experience.

## How to Run

1. Make sure you have Python installed.
2. Run the main application:
   ```bash
   python main.py
   ```

## File Structure

- `main.py`: Entry point for the application.
- `sudoku_gui.py`: Contains the GUI implementation.
- `sudoku_logic.py`: Core Sudoku solving and generation logic.
- `test_solver.py`: Testing script for the Sudoku solver.
- `test_generator.py`: Testing script for the puzzle generator.

## Project Context
This project was developed for the Design and Analysis of Algorithms (DAA) coursework.
